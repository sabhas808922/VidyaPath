# VidyaPath :
It is an inclusive, mobile-first EdTech platform designed for underserved communities in India, including:

Rural learners
Users with low-end devices (₹5000 phones)
Differently-abled users:
Visually impaired
Hearing impaired
Cognitively disabled

The platform must solve:

- Lack of accessibility in existing EdTech apps
- High device and internet requirements
- Fragmented learning across multiple apps
- Paid-first models blocking access

## Core Vision

The product must be:

- Lightweight
- Offline-first (works without internet)
- Accessibility-first (NOT optional)
- Free-first (no paywall in MVP)
- Optimized for low bandwidth + low RAM devices
